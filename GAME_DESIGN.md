# 蛋宝世界 (DANBO World) — 游戏设计文档

## 概述
蛋宝世界是一款基于 Three.js r160 的 3D 卡通障碍赛跑 + 开放城市探索游戏，灵感来自蛋仔派对。玩家操控蛋形角色在多个主题城市中探索、竞速、互动。

- 引擎: Three.js r160 (CDN `<script>` 标签加载，unpkg 备用)
- 渲染风格: `MeshToonMaterial` 卡通渲染
- 音频: Web Audio API 程序化生成（无音频文件）
- 国际化: 4 语言（简体中文、繁体中文、日语、英语），自动检测浏览器语言
- 版本格式: `vYYYYMMDD.N`（N = 当日构建号）
- 部署: GitHub Pages，自定义域名 `eggy.ff18.com`

## 传送系统

### 水管传送
- Mario 3D World 风格透明管道
- 地面城市间互通（0-4），月球不可直达
- 进入后飞行动画（贝塞尔曲线弧线）
- 飞行途中重建目标城市

### 巴别塔
- 地面城市中心附近，与云阶梯偏移8-10单位
- 4面门（东南西北）
- 电梯上下：地面 ↔ 云中界
- 地震事件触发升起动画
- 挡住相机视线时变透明（加入 cityBuildingMeshes）
- NPC 也会使用巴别塔上下

### 月球传送
- 云中界顶部的特殊管道
- 确认对话框后传送
- 到达后出现在冯·布劳恩城内(-200,3,0)

### 地球返回
- 月面冯·布劳恩城内的返回传送门
- 返回玩家来时的城市（记录 _prevCityStyle）

## 操作系统

### 键盘
| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动 |
| 空格 | 跳跃（点按普通跳，长按0.3秒后蓄力跳） |
| F | 抓取/投掷（点按抓取，再按投掷，长按蓄力投掷） |
| Shift | 冲刺（未抓取时） |
| Enter | 打开聊天输入 |
| 鼠标滚轮 | 缩放 |
| 右键拖动 | 月球视角旋转 |
| Y/Enter | 确认传送 |
| N/ESC | 取消传送 |

### 移动端
- 虚拟摇杆（左侧）
- 跳跃按钮、抓取按钮
- 双指缩放

### 抓取/投掷机制
- F 键抓取最近的 NPC/道具/障碍物
- 再按 F 普通投掷
- 长按 F 超过0.3秒进入蓄力模式，松开蓄力投掷
- 同一次按键不会同时抓取+投掷
- 抓着东西时不能冲刺
- 抓着东西时不能蓄力跳（只能小跳，JUMP_FORCE×1.2）
- NPC 投掷力度: force=0.4, vy=0.15, throwTimer=80
- 蓄力投掷距离明显大于普通投掷

### 蓄力跳
- 长按空格0.3秒后进入蓄力模式
- 蓄力进度条显示在画面上
- 蓄力越满跳越高（最大 JUMP_FORCE×4.0）
- 蓄力满后有限时间窗口，超时自动取消
- 蓄力跳产生屁股烟雾特效
- 抓着东西时禁用蓄力跳

### 冲刺 & 音速旋转冲刺
- Shift 键冲刺（2倍速）
- 冲刺蓄力释放 → 音速旋转冲刺（4倍速 = 2倍冲刺速）
- 旋转冲刺可操控方向，贴地行进
- 进度条显示冲刺能量
- 旋转冲刺可撞飞 NPC 和道具（Y距离限制1.5）
- 旋转冲刺不触发传送门/水管/巴别塔门
- 被击晕时旋转冲刺自动取消
- NPC 也会使用旋转冲刺

## 物理系统
- 重力: GRAVITY = 0.018
- 跳跃力: JUMP_FORCE = 0.28
- 移动加速: MOVE_ACCEL = 0.016
- 最大速度: MAX_SPEED = 0.22
- 摩擦力: FRICTION = 0.92
- 城市边界: 空气墙反弹
- 投掷物理: 弹跳次数、阻力、着陆判定
- 建筑碰撞: 可站在屋顶
- 云平台碰撞: 可站在云上
- AT力场碰撞: 圆形护盾，门口可通过

## 击晕系统
- 基于冲击力的击晕时长
- SF2 风格 4 级击晕视觉（按严重程度分级）:
  - 轻微: 小星星
  - 中等: 旋转星星
  - 严重: 小鸟环绕
  - 极重: 小鸭子环绕
- 旋转冲刺被击晕时自动取消

## 聊天系统
- PSOBB 漫画式说话气泡
- Enter 键打开输入框
- 输入消息后显示在角色头顶的 3D 精灵气泡
- 气泡持续5秒后淡出
- NPC 随机说话（低概率）
- 支持4语言的 NPC 台词

## NPC AI 系统
NPC 有多种行为状态，随机切换：

| 状态 | 行为 |
|------|------|
| wander | 随机漫步，偶尔冲刺/跳跃 |
| idle | 原地站立，偶尔张望/小跳 |
| chase | 追逐最近的蛋 |
| flee | 逃离最近的蛋 |
| dance | 原地旋转跳舞 |
| circle | 绕圈行走 |
| babel | 走向巴别塔，使用电梯上下云中界 |
| spinDash | 蓄力后旋转冲刺，撞飞附近的蛋 |

- NPC 会抓取其他蛋（低概率）
- NPC 会抓取并投掷道具
- NPC 蓄力跳可跳上屋顶
- NPC 使用巴别塔在地面和云中界之间移动

## 月球战斗系统 (MS 机体)

### 机体类型
- 标准 MS（光束步枪/导弹）
- SDF-1（大型舰船）
- 杰特拉帝巡洋舰（大型舰船）
- 各阵营: EFSF、Zeon、UN Spacy、Zentradi

### 战斗机制
- MS 在战场区高速飞行，AI 寻路
- 光束武器: 直线射击，命中护盾产生 AT 力场特效
- 导弹: 追踪射击，带烟雾尾迹
- MS 被击毁后大爆炸 + 碎片散落，定时重生
- 随机战斗伤害爆炸
- 漏斗武器（浮游炮）

### 城市防护
- AT 力场护盾阻挡 MS 进入
- 光束/导弹命中护盾产生 AT 力场视觉特效
- 城市内蛋不受光束/导弹伤害
- 城市内爆炸视觉完全隐藏
- 城市内战斗声音衰减至 8%
- MS 群体也被隐藏（玩家在城市内时）

## 国际化 (i18n)
- 4 语言: 简体中文(zhs)、繁体中文(zht)、日语(ja)、英语(en)
- 自动检测浏览器语言
- 语言下拉菜单（向下弹出）
- 支持 Auto 模式（跟随浏览器）
- 所有 UI 文本、角色名、城市名、赛道名、提示文字均已翻译
- 月面都市名称: 冯·布劳恩 / 格拉纳达（4语言）
- 挣扎提示、聊天占位符等均已翻译

## 道具系统
- 城市内可抓取的道具: 氧气罐、箱子、桶、天线等
- 投掷后弹跳、翻倒
- 撞墙反弹（空气墙）
- 着陆后可重新抓取
- 投掷道具撞到 NPC 产生击退 + 击晕

## 金币系统
- 城市内散布金币（地面城市90个，月球200个）
- 竞速中收集金币
- NPC 被击中掉落偷取的金币
- 金币散落物理动画
- 城市边界反弹

## 技术细节
- Three.js r160 via CDN（非 ES 模块）
- 无 ES6 `\u{XXXXX}` unicode 转义
- 无模板字符串在 I18N 中
- `/* global THREE */` 声明
- `server.py` 本地测试服务器 (0.0.0.0:8080)
- GitHub Pages 部署，HTTPS
- 每次修改后 `node -c game.js` 语法检查
